# Règles de validation — Conformité globale d'un dossier CEE

> **Principe absolu** : Pour valider la conformité totale du dossier, TOUS les éléments ci-dessous doivent être conformes. Si un seul élément n'est pas valide, le dossier revient NON CONFORME.
>
> - Le professionnel ayant réalisé les travaux est défini comme le **professionnel installateur**.
> - Le professionnel défini comme installateur **ne peut jamais** être considéré comme un MOE.

---

## 1. Logique globale — Documents obligatoires

Un dossier ODICEE est composé, au minimum, de :
- **Un document d'engagement** (OS, AE, devis, bon de commande)
- **Une preuve de réalisation** (facture, DGD, PV de réception + DPGF/devis)
- **Une attestation sur l'honneur** (parties A, B, C et BS si logements conventionnés)

Documents supplémentaires obligatoires selon le contexte :
- **Certificat RGE** : si les travaux nécessitent que le professionnel soit RGE
- **Tableau de répartition** (Excel ou PDF) : si les travaux nécessitent une répartition d'isolation

> ⚠️ Au moins un document du dossier doit contenir la ou les **adresses précises** des travaux.
> **Adresse précise** = numéro + rue + code postal + ville. Le "lieu des travaux sans adresse exacte" (nom de résidence, nom de rue) ne sert qu'à la vérification de cohérence entre documents, pas à satisfaire cette exigence.

### Règle sous-traitance
Si un sous-traitant est mentionné dans le dossier :
- Il faut vérifier s'il a réalisé les travaux valorisés
- Si oui → une **déclaration de sous-traitance (DC4) conforme** est obligatoire
- Alternative : une facture ou un devis du sous-traitant décrivant les travaux peut remplacer la DC4 pour prouver qu'il n'a pas réalisé les travaux valorisés

---

## 2. Validation engagement
Appliquer les **règles de la preuve d'engagement** pour le type de document concerné (OS, AE, devis, BC).

## 3. Validation réalisation
Appliquer toutes les **règles de la preuve de réalisation** pour le type de document concerné, ainsi que tous les **éléments techniques** de la fiche BAR/BAT concernée.

## 4. Validation RGE
- Si la date d'engagement est dans la période d'une fiche BAR nécessitant le RGE → le certificat RGE doit respecter les règles de certification.
- Le **SIRET du professionnel** permettant la vérification du RGE doit se trouver sur la preuve de réalisation.
- S'il y a un sous-traitant : le SIRET à vérifier se trouve sur la DC4 → la DC4 doit être conforme.

## 5. Validation attestation sur l'honneur
Appliquer les règles de l'attestation sur l'honneur (parties A, B, C, BS).

---

## 6. Validation cohérence engagement ↔ réalisation

Un **lien fort** doit obligatoirement être établi entre le document d'engagement et le document de réalisation.

**Définition officielle du lien fort** — la cohérence est CONFORME si :
- (**Numéro de marché ET Adresse des travaux** concordants) **OU**
- (**Prix HT (€)** concordant)

> Tolérance prix : un écart de **1 centime maximum** est toléré s'il provient d'un arrondi. Au-delà, l'écart doit être justifié ou la cohérence n'est pas établie.

> Tout élément analysé sur les documents marché concernant les travaux doit être **cohérent entre eux**, que ce soit sur un même document ou entre les documents marché.

---

## 7. Validation documents annexes

Si la fiche impose un document annexe, il doit contenir les éléments nécessaires décrits dans les règles autres documents.

Documents annexes possibles :
- Tableau de répartition d'isolation
- Tableau de répartition des surfaces habitables
- Étude de dimensionnement
